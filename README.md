# Admitere Medicină – dr. Zamfir Alexandru, Versiunea 2.0

Această versiune păstrează funcțiile testului și adaugă:

- logo SVG original;
- intro animat;
- aspect premium albastru închis și auriu;
- borderou compact;
- avertizări la 15 și 5 minute;
- predare automată la expirarea timpului;
- punctaj, greșeli, necompletate și barem complet.

## Instalare pe GitHub

Înlocuiți în repository fișierele:

- `index.html`
- `style.css`
- `app.js`
- `barem.js`

și încărcați folderul:

- `assets/logo.svg`

GitHub Pages va actualiza automat același link.
